1. CPP
1) Пользователь вводит имя файла и количество частей, на которое этот файл нужно разбить. Последняя часть может
быть меньшего размера. Если файл сильно маленький, то отказ в работе.
2) Пользователь вводит количество частей и имена файлов. Программа объединяет их в один файл

3) ЗАДАЧА на +1 балл к экзамену
Программа позволяет выполнять следующие команды языка SQL:
- create table [NAME] ([field1] [type] primary key, [field2] [type], [field3] [type])
[type] может быть int, varchar(n)
field - имеет максимальную длину в 30 символов

- select * from [NAME]

- insert into [NAME] ([field1], [field2], [field3])
values 
(val1, val2, val3)
